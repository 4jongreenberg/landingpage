Getting Started

Include the start.js file with an id of webplate before your closing body tag.

<body style="display:none;">
    /* Your content goes here */
    <script id="webplate" src="webplate/start.js"></script>
</body>

Load Your Project Files

CSS and Javascript is included via a configuration file found at webplate/project/config.json.

{
    "project": {
        "css": ["welcome.css", "theme.css"],
        "js": ["min/welcome.min.js"],
    }
}

A comma delimited list allows you to load more than one file at a time. The full path is not required and is automatically maped to the project directory. You can also load different files on different pages. Want to know more?
